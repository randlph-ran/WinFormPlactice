## プロジェクト概要 <a id="project-overview"></a>

| 項目 | 内容 |
|------|------|
| **タイトル** | ファミレスメニューシミュレータ |
| **一言で説明すると** | ファミレスにある注文タブレットの挙動をシミュレートしたアプリ |
| **制作の目的** | WinFormでの画像やデータの読み込みと出力、サウンドの再生などを学ぶ |
| **特徴・工夫点** | 料理の注文を確定すると、レシートのようなテキストファイルが出力されるようにした　多くの料理はcsvファイルでデータを管理しているのでちょっとの調整だけで無限に料理が追加ができる |
